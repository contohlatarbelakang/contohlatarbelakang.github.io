Melampirkan File

Di ponsel atau tablet Android, buka aplikasi Gmail Gmail.

Tap Tulis Tulis.

Tap Lampirkan Lampirkan.

Ketuk Lampirkan file atau Sisipkan dari Drive.

Pilih file yang akan dilampirkan.

Catatan: Untuk beberapa jenis file, seperti file ZIP, Anda perlu mendownload file manager dari Google Play sebelum dapat melampirkannya.

23. bagaimana langkah langkah mengirim melalui lampiran file?


1. Aktifkan internet dengan menggunakan salah satu browser, seperti Micrososft Internet Explorer atau Mozilla Firefox, atau browser lainnya. Buka situs penyedia layanan E-mail, misalnya di Yahoo, Gmail, atau Hotmail. Setelah itu isikan ID (alamat e-mail) dan password (kata sandi) yang Anda miliki, ketika membuat account e-mail.

Gambar Pengisian ID dan password di Yahoo

2. Klik tombol Sign In untuk membuka account e-mail tersebut.

3. Untuk mengirim e-mail dengan file lampiran, klik link atau tombol New atau Compose,

Gambar Form Pengiriman Pesan di Yahoo

4. Isilah bagian-bagian berikut:

- To : Isikan dengan alamat e-mail yang dituju

- Cc : Isikan dengan alamat e-mail lain yang dituju (jika ada)

- BCc : Isikan dengan alamat e-mail lain lagi yang dituju (jika ada)

- Subject : Isikan dengan perihal atau judul e-mail
6.Setelah itu ketikkan pesan Anda.

7. Setelah selesai klik tombol Attach Files untuk menyertakan file lampiran dalam email, sehingga muncul tampilan kotak dialog Upload File.

8. Pilih lokasi file Anda di kotak Look in, lalu pilih file yang Anda ingin lampirkan, setelah itu klik tombol Open.

9. Tunggu sejenak sampai proses upload file selesai

10. Setelah proses upload file selesai, tekan tombol Send untuk mengirim e-mail dengan file lampirannya.

11. Tunggu sejenak sampai dengan muncul pesan “Massage Sent”, hal ini menunjukkan bahwa pesan Anda telah berhasil dikirim.

12. Apabila tidak ada lagi keperluan dengan account e-mail Anda, jangan lupa untuk menekan tombol Sign Out.

Semoga membantu :)

38. 1.apa yang dimaksud iklan pemberitahuan?2.contoh iklan pemberitahuan?3.contoh kata kata iklan pemberitahuan?​


1. Iklan pemberitahuan adalah jenis iklan yang bertujuan untuk memberikan informasi kepada khalayak tentang suatu peristiwa, acara, penawaran, atau perubahan yang relevan. Iklan ini biasanya digunakan sebagai cara untuk menyampaikan pesan penting kepada target audiens.

2. Contoh iklan pemberitahuan:

a) Iklan pemberitahuan tentang acara konser: "Bertunanganlah dengan musik di konser eksklusif band ABC! Tanggal 15 Agustus di Stadion Utama, jangan lewatkan kesempatan ini untuk merasakan kegembiraan dan keseruan!"

b) Iklan pemberitahuan tentang penutupan sementara toko: "Perhatian pelanggan setia! Kami menginformasikan bahwa toko kami akan tutup sementara mulai tanggal 1 hingga 15 September karena renovasi. Kami mohon maaf atas ketidaknyamanan ini dan akan kembali memberikan pelayanan terbaik setelah renovasi selesai."

3. Contoh kata-kata dalam iklan pemberitahuan:

a) "Perhatian"

b) "Penting"

c) "Informasi penting"

d) "Pengumuman"

e) "Diberitahukan kepada semua"

f) "Segera"

g) "Mohon dicatat"

h) "Jangan lewatkan"

i) "Sampai ketemu di"

j) "Simpan tanggalnya"

40. Bagaimanacara melampirkan sebuah file​


Melampirkan file ke pesan

1. Membuat pesan, atau untuk pesan yang sudah ada, klik Balas, Balas Semua, atau Teruskan.

2. Di jendela pesan, di tab Pesan, di grup Sertakan, klik Lampirkan File.

3. Telusuri dan klik file yang ingin dilampirkan, kemudian klik Sisipkan.

Melampirkan file atau menyisipkan gambar dalam pesan email Outlook
Outlook untuk Office 365 Selengkapnya...
Mudah untuk melampirkan gambar, file, kontak, email dan item lainnya ke pesan Outlook. Outlook terus melacak dokumen yang baru saja dikerjakan, baik yang disimpan di komputer Anda maupun disimpan di OneDrive (cloud saja). Selain itu, di mana pun dokumen disimpan, Outlook memungkinkan Anda langsung memilih apakah akan mengirimkan dokumen sebagai lampiran biasa atau mengunggahnya ke OneDrive dan membagikan tautannya. Anda juga dapat mengatur izin pada file yang dikirimkan dari OneDrive, memungkinkan penerima pesan untuk melihat dan mengeditnya atau membagikannya dengan orang lain.

Untuk mengatasi masalah saat melampirkan file, lihat Mengatasi masalah lampiran.
