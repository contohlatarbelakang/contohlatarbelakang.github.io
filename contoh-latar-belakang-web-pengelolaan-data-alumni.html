21. Web atau situs web memiliki halaman web yang berisi data dan informasi yang tersimpan di web server dalam bentuk​


﻿﻿Web atau situs web biasanya terdiri dari halaman-halaman web yang berisi data dan informasi dalam bentuk teks, gambar, video, dan elemen-elemen multimedia lainnya. Halaman-halaman web ini disimpan di server web dalam bentuk file HTML, CSS, JavaScript, dan file media lainnya.

﻿File HTML (Hypertext Markup Language) adalah dasar dari semua halaman web dan digunakan untuk membangun struktur halaman web. File CSS (Cascading Style Sheets) digunakan untuk memperindah tampilan halaman web dengan mengatur warna, font, dan layout. Sedangkan file JavaScript digunakan untuk membuat halaman web interaktif dan dinamis.

﻿Selain itu, halaman web juga dapat menyimpan data dan informasi dalam bentuk file media seperti gambar, video, dan audio. File-file media ini disimpan di server web dan dapat diakses oleh pengguna melalui halaman web.

﻿Secara umum, semua data dan informasi yang terdapat di halaman web disimpan di server web dalam bentuk file yang kemudian diakses oleh pengguna melalui browser web.

34. diagram lingkaran di bawah ini menyatakan data alumni suatu smk yg telah bekerja diberbagai bidang.jika jumlah alumni tersebut 1200 orang maka jumlah yg berwirausaha adalah


Diagram lingkaran di bawah ini menyatakan data alumni suatu smk yg telah bekerja diberbagai bidang.jika jumlah alumni tersebut 1200 orang maka jumlah yg berwirausaha adalah 240 orang

Karena soal tidak dicantumkan diagram lingkarannya, maka saya mengambil diagram lingkaran dari tugas lainnya. Soal berikut ini adalah soal pengolahan data dan penyajian data

Untuk mencari banyak alumni yang berwirausaha maka kita harus mencari sudut-sudut yang belum diketahui terlebih dahulu

Diagram lingkaran (diagram pai) adalah sebuah grafik statistik berbentuk lingkaran yang dibagi menjadi irisan-irisan untuk menggambarkan proporsi numerik

Besar sudut 1 diagram lingkaran adalah 360° atau 100%

Pembahasan :

Diketahui :

Jumlah alumni = 1.200 orang

Staf wirausaha 40%

Perniagaan 20%

Pegawai KUD 10%

Perpajakan 5%

Lainnya 5%

Ditanya :

Jumlah orang yang berwirausaha ?

Dijawab :

Pertama-tama kita harus mencari persentase orang yang melakukan wirausaha terlebih dahulu

Jumlah seluruh alumni = 100%

Wira usaha = 100% - (40% + 20% + 10% + 5% + 5%) = 20%

Jumlah seluruh alumni = 1.200 orang

Jumlah alumni yang berwirausaha = [tex]\frac{20}{100}[/tex] x 1.200 = 240 orang

∴ Jadi jumlah alumni yang berwirausaha adalah 240 orang

36. Diagram di samping menunjukkan kondisi alumni SMK tahun 2016 diperoleh data bahwa sebanyak 70% alumni tsb bekerja dan seperempat dari seluruh alumni tsb memilih untuk kuliah, jika 15 orang alumni menganggur, banyak lulusan SMK yang tidak menganggur ada orang. Lulusan SMK l I Bekerja 0 Kuliah I Menganggur


Jumlah alumni = jumlah yg bekerja + jumlah yg kuliah + jumlah yg menganggur

100% alumni = 70% + 25% + 15 orang
15 orang = (100-70-25)% = 5%

jumlah yg menganggur = 5%

jumlah yg tidak menganggur = 95%

% yg menganggur = (jumlah orang menganggur / jumlah seluruh alumni) x 100%

5% = (15 org / total alumni) x 100%
0,05 = 15 org / total alumni
total alumni = 15 / 0,05
total alumni = 300 orang

total yg tidak menganggur = 95% x 300 orang = 285 orang
