11. Buatlah kode html dan css agar menjadi seperti di gambar


Penjelasan:

Untuk membuat tata letak dengan elemen-elemen yang mengikuti deskripsi yang kamu berikan, kamu dapat menggunakan HTML dan CSS seperti ini:

```html

<!DOCTYPE html>

<html lang="en">

<head>

<meta charset="UTF-8">

<meta name="viewport" content="width=device-width, initial-scale=1.0">

<style>

.container {

width: 100%;

overflow: hidden; /* Untuk menangani float */

}

.atas {

background-color: lightblue;

}

.kiri {

float: left;

width: 25%;

background-color: lightgreen;

}

.tengah {

float: right;

width: 25%;

background-color: lightcoral;

}

.kanan {

float: right;

width: 25%;

background-color: lightpink;

}

.bawah {

clear: both; /* Menghindari elemen mengikuti float */

background-color: lightyellow;

}

</style>

</head>

<body>

<div class="container">

<div class="atas">

<h1>Sisi Atas (tanpa float)</h1>

</div>

<div class="kiri">

<p>Sisi Kiri (float left)</p>

</div>

<div class="tengah">

<p>Sisi Tengah (float right)</p>

</div>

<div class="kanan">

<p>Sisi Kanan (float right)</p>

</div>

<div class="bawah">

<p>Sisi Bawah (tanpa float)</p>

</div>

</div>

</body>

</html>

```

Dalam kode di atas, saya menggunakan float untuk mengatur sisi kiri, sisi tengah, dan sisi kanan. Elemen-elemen tersebut memiliki width 25% untuk memastikan bahwa mereka berada di sisi yang sesuai. Elemen sisi atas dan sisi bawah tidak menggunakan float, dan elemen sisi bawah menggunakan `clear: both` untuk menghindari elemen mengikuti float.

25. jelaskan pengertian HTML dan CSS!​


HTML adalah, (HyperText Markup Language) sebuah bahasa standar yang digunakan oleh browser Internet untuk membuat halaman dan dokumen pada sebuah Web yang kemudian dapat diakses dan dibaca layaknya sebuah artikel. Cascading Style Sheet (CSS) merupakan salah satu bahasa pemrograman web untuk mengendalikan beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam.- HTML (Hypertext Markup Language), yaitu bahasa markup standar untuk membuat dan menyusun halaman dan aplikasi web. Kita bisa menggunakan bahasa markup ini untuk bagian paragraf, heading, maupun link pada suatu web page.

- CSS (Cascading Style Sheet) dan biasanya digunakan untuk mengatur tampilan elemen yang tertulis dalam bahasa markup, seperti HTML. CSS berfungsi untuk memisahkan konten dari tampilan visualnya di situs.

Perbedaan utama antara HTML vs. CSS adalah: HTML menentukan konten dan struktur halaman web, dan CSS menentukan desain dan tampilan elemen HTML. File CSS dapat digunakan dalam file HTML, tetapi file CSS bukanlah halaman web yang akan ditampilkan sendiri.

28. jelaskan perbedaan antara tag di HTML dan tag di CSS!


Jawaban:

HTML adalah singkatan dari HyperText Markup Language , yaitu markup language yang digunakan untuk membuat suatu halaman web, sehingga HTML dapat menampilkan berbagai informasi baik tulisan maupun gambar pada sebuah web browser.

CSS adalah singkatan dari Cascading Style Sheet, yaitu suatu aturan yang digunakan untuk mengatur tampilan maupun format dari suatu halaman web, sehingga tampilan web akan menjadi lebih teratur.

--semoga membantu--

Jawaban:

HTML biasanya menggunakan tag-tag tertentu yang berada dalam tanda kurung <>. Sebagian besar tag pada HTML adalah tag yang berpasangan, contoh : jika kita menggunakan <head> harus ditutup dengan </head>, atau jika kita menggunakan <p> harus ditutup dengan </p>. Tapi ada juga beberapa tag yang tidak berpasangan, contohnya adalah <img>.

Sedangkan untuk CSS ada 3 cara penulisan, yakni internal, inline dan eksternal.

Contoh penulisan inline :

<h1 style="color:red">Judul</h1>

Internal CSS diletakkan di dalam tag <style>, dan biasanya tag <style> ditempatkan pada bagian tag <head>  

Contoh penulisan internal :

<style type="text/css">

.box{

 background: blue;

 padding: 10px;

}

</style>

Sementara untuk eksternal, antara file html dan css dipisah,berikut cara menghubungkan file css dengan html :  

<link rel="stylesheet" type="text/css" href="file.css">
