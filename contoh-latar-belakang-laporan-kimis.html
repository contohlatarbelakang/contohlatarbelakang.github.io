Penjelasan dengan langkah-langkah:

16. Misalkan:

RH = harga 1 buah roti kacang hijau

RK = harga 1 buah roti kismis

RM = harga 1 buah roti mentega

Penyelesaian:

2RH + RK + 3RM = Rp. 20.000 … persamaan (1)

2RH + 2RK + 5RM = Rp. 30.000 … persamaan (2)

5RH + 3RK + 2RM = Rp. 29.000 … persamaan (3)

# METODE ELIMINASI

- Eliminasi persamaan (1) dan (2), menghilangkan RH:

2RH + RK + 3RM = 20.000

2RH + 2RK + 5RM = 30.000   -

         -RK - 2RM = -10.000 |dikali (-1)| <=> RK + 2RM = 10.000 … persamaan (4)

- Eliminasi persamaan (2) dan (3), menghilangkan RH:

2RH + 2RK + 5RM = Rp. 30.000 |dikali 5|<=> 10RH + 10RK + 25RM = 150.000

5RH + 3RK + 2RM = 29.000      |dikali 2|<=> 10RH + 6RK + 4RM = 58.000 -

                                           4RK + 21RM = 92.000 … persamaan (5)

- Eliminasi persamaan (4) dan (5), mencari nilai RK:

RK + 2RM = 10.000      |dikali 21|<=> 21RK + 42RM = 210.000

4RK + 21RM = 92.000   |dikali 2|<=> 8RK + 42RM = 184.000    -

                        13RK           = 26.000

                            RK             = 26.000/13

                            RK            = 2.000

     

# METODE SUBSTITUSI

- Substitusi persamaan (4) mencari nilai RM:

 RK + 2RM = 10.000    

 2.000 + 2RM = 10.000      

    2RM = 10.000 – 2.000    

    2RM = 8.000    

    RM = 8.000/2    

    RM = 4.000    

         

- Substitusi persamaan (2) mencari nilai RH:

2RH + 2RK + 5RM = 30.000

2RH + 2(2.000) + 5(4.000) = 30.000

2RH + 4.000 + 20.000 = 30.000

2RH + 24.000 = 30.000

2RH = 30.000 – 24.000

2RH = 6.000

2RH = 6.000/2

RH = 3.000

Jadi, jika membeli 5 buah roti kacang hijau adalah 5 x 3.000 = 15.000 dan 4 buah roti mentega adalah 4 x 4.000 = 16.000, maka kita harus membayar sebesar 15.000 + 16.000 = 31.000
