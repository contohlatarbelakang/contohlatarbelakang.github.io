5. Perintah yang digunakan untuk mengetahui IP address yang digunakan pada Briker adalah​


Konfigurasi IP Address

konfigurasi ip address adalah hal yang utama, karena briker merupakan server yang nantinya akan diakses oleh client, maka kita harus membuat briker bisa diakses oleh client yang ingin tersambung.

Disini kalian harus berfikir bagaimana client bisa tersambung nantinya, karena kita menginstall nya di virtualbox, kita bisa melakukan berikut yang paling sederhana :

   menggunakan adapter host-only, jadi yang terhubung hanya komputer vm, briker, dan pc fisik (host) kita saja

   menggunakan bridged adapter, jadi dengan ini kita bisa share secara bridge ke suatu jaringan misalnya kita share melalui WiFi agar teman kita atau laptop / hp lain bisa mengakses Briker kita asal dalam jaringan WiFi yang sama.

Dengan dua kasus sederhana diatas, saya akan mneggunakan cara kedua yaitu mem-bridge ke adapter WiFi laptop saya, kenapa ? karena nantinya saya ingin menguji coba dengan hp dan laptop teman saya yang lain. Jadi disini pastikan kalian punya koneksi wifi entah dari Access Point atau kalian hotspot dari hp kalian.

Dengan bridge adapter kita juga bisa membridge ke LAN adapter di laptop kita agar nantinya bisa dishare ke jaringan menggunakan switch atau peer-to-peer misalnya. Jadi pastikan kalian sudah memahami hubungan antar Network Modes di VirtualBox ini.

Jika kalian belum paham apa itu Host-only, Bridge Adapter dan kalian bingung bagaiamana memilih yang tepat, kalian bisa mempelajari nya di tutorial berikut :

Mengenal Macam dan Fungsi Network Modes di VirtualBox

Oke untuk tutorial kali ini, saya akan menggunakan kedua cara diatas, yaitu menggunakan Host-only dan menggunakan Bridge Adapter yang saya arahkan ke WiFi agar bisa diakses oleh laptop atau hp teman saya. Jadi kalian harus punya sambungan wifi atau lan di laptop kalian

Pembahasan

VoIP adalah teknologi komunikasi suara melalui IP (Internet Protocol) melalui jaringan paket. Teknologi ini bekerja dengan mengubah audio tertentu menjadi format digital yang dapat ditransmisikan melalui jaringan IP. Menurut definisi, Voice over Internet Protocol (VoIP) adalah teknologi komunikasi yang memiliki cara transmisi tidak hanya suara dan audio, tetapi juga konten lain seperti gambar dan video. Via Koneksi Internet VoIP sendiri merupakan media untuk panggilan jarak jauh melalui Internet. Media Internet membuat panggilan lebih hemat biaya daripada telepon tradisional, terutama untuk panggilan jarak jauh antar negara.

13. sebutkan perintah untuk masuk interfaces dan konfigurasi ip pada briker !​


Jawaban:

Berikut adalah beberapa perintah untuk masuk ke interfaces dan konfigurasi IP pada Briker:

1. Masuk ke mode konfigurasi

Untuk masuk ke mode konfigurasi, gunakan perintah berikut:

 

enable

 

2. Masuk ke konfigurasi interfaces

Untuk masuk ke konfigurasi interfaces, gunakan perintah berikut:

 

configure terminal

 

3. Pilih interface yang akan dikonfigurasi

Untuk memilih interface yang akan dikonfigurasi, gunakan perintah berikut:

 

interface <nama_interface>

 

4. Konfigurasi IP pada interface

Untuk mengkonfigurasi IP pada interface, gunakan perintah berikut:

 

ip address <alamat_IP> <subnet_mask>

 

5. Aktifkan interface

Setelah konfigurasi selesai, aktifkan interface dengan menggunakan perintah berikut:

 

no shutdown

 

6. Simpan konfigurasi

Setelah selesai melakukan konfigurasi, pastikan untuk menyimpan konfigurasi dengan menggunakan perintah berikut:

 

write memory

 

Perintah-perintah di atas dapat bervariasi tergantung pada jenis perangkat Briker yang digunakan dan versi perangkat lunak yang digunakan. Pastikan untuk merujuk pada dokumentasi resmi Briker atau manual pengguna untuk informasi lebih lanjut mengenai konfigurasi interfaces dan IP.

Untuk masuk ke antarmuka (interface) dan mengkonfigurasi alamat IP pada router (briker) menggunakan perangkat lunak seperti Cisco IOS, Anda dapat menggunakan perintah berikut:

Masuk ke mode konfigurasi:

enable

Masuk ke konfigurasi antarmuka yang diinginkan, misalnya, antarmuka GigabitEthernet0/0:

configure terminal

interface GigabitEthernet0/0

Konfigurasi alamat IP dan subnet mask:

ip address <alamat_IP> <subnet_mask>

Aktifkan antarmuka:

no shutdown

Simpan konfigurasi:

end

write memory

Pastikan untuk mengganti <alamat_IP> dan <subnet_mask> dengan nilai yang sesuai untuk jaringan Anda.

22. sebutkan cara kerja briker?​


Briker adalah sebuah server VoIP (Voice over Internet Protocol) yang berbasis open source. Briker dapat digunakan untuk membuat sistem telepon internal, konferensi audio, dan video, serta layanan telepon ke PSTN (Public Switched Telephone Network).

Cara kerja Briker adalah sebagai berikut:

Instalasi Briker

Briker dapat diinstal di berbagai platform, seperti Linux, Windows, dan macOS. Untuk menginstal Briker, Anda dapat mengikuti instruksi yang tersedia di situs web Briker.

Konfigurasi Briker

Setelah Briker diinstal, Anda perlu melakukan konfigurasi dasar. Konfigurasi dasar ini meliputi pengaturan IP address, port, dan username/password.

Pembuatan Extension

Extension adalah nomor telepon internal yang dapat digunakan untuk melakukan panggilan telepon. Untuk membuat extension, Anda dapat menggunakan antarmuka web Briker atau melalui CLI (Command Line Interface).

Panggilan Telepon

Setelah extension dibuat, Anda dapat melakukan panggilan telepon ke extension lain atau ke PSTN. Untuk melakukan panggilan telepon, Anda dapat menggunakan antarmuka web Briker, melalui CLI, atau melalui perangkat lunak VoIP client.

Secara lebih rinci, cara kerja Briker dapat dijelaskan sebagai berikut:

Briker menggunakan protokol SIP (Session Initiation Protocol) untuk berkomunikasi dengan perangkat lunak VoIP client dan perangkat keras telepon IP.Briker menggunakan protokol RTP (Real-time Transport Protocol) untuk mengirimkan data suara antar perangkat lunak VoIP client dan perangkat keras telepon IP.Briker menggunakan protokol IAX (Inter-Asterisk Exchange) untuk berkomunikasi dengan server VoIP lain.

Berikut adalah beberapa fitur Briker:

Open sourceGratisDapat digunakan di berbagai platformMemiliki fitur yang lengkapMudah digunakan

Briker merupakan server VoIP yang cocok untuk berbagai kebutuhan, baik untuk penggunaan pribadi maupun bisnis. Briker menawarkan fitur yang lengkap dan mudah digunakan, serta dapat digunakan di berbagai platform.

28. perintah untuk me-restart briker​


Untuk merestart perangkat Bricker, Anda dapat mencoba beberapa perintah berikut pada Command Line Interface (CLI) atau SSH (Secure Shell):

1. Restart Perangkat:

- sudo reboot

- sudo shutdown -r now

2. Restart Layanan atau Servis Bricker:

- sudo service bricker restart

- sudo systemctl restart bricker

Pastikan Anda memiliki hak akses administratif (superuser/root) saat menggunakan perintah ini. Selain itu, perintah yang tepat akan tergantung pada sistem operasi dan konfigurasi spesifik perangkat Bricker yang Anda gunakan. Pastikan untuk merujuk pada dokumentasi resmi atau panduan pengguna perangkat Bricker Anda untuk petunjuk lebih lanjut atau perintah yang disarankan.

39. apa saja contoh sistem operasi beserta versinya??


Contoh sistem operasi :
1. Unix

Unix adalah sistem operasi tertua yang dibuat dan dioperasikan hanya menggunakan Text Mode artinya sistem tidak menampilkan mode grafis (gambar)

2. Apple Mac OS

Mac OS adalah sistem operasi yang dibuat oleh perusahaan bernama Apple yang didirikan oleh Steve Jobs, Mac OS merupakan OS berbayar yang dijual bersama dengan perangkat kerasnya.

Mac OS merupakan OS turunan Unix, versi Mac OS antara lain : Cheetah, Puma, Jaguar, Panther, Tiger, Leopard, Snow Leopard, Lion

3. Microsoft Windows

Windows adalah sistem operasi yang dibuat oleh perusahaan yang bernama Microsoft yang didirikan oleh Bill Gates, Windows merupakan OS yang diturunkan dari DOS (Disk Operating System)

Versi Microsoft Windows :

windows 95, 98, 2000, ME, XP, Vista, 7, 8,10

Sistem operasi windows merupakan sistem operasi berlisensi yang mengharuskan pemakainya harus membeli/ membayar lisensi tersebut

5. Android

Android adalah sistem operasi yang diturunkan dari linux awalnya dibuat oleh perusahaan yang bernama Android inc. yang akhirnya dibeli oleh perusahaan Google, Android merupakan sistem operasi yang digunakan pada perangkat mobile seperti : Smartphone, Tablet, Phablet, dsb.

Versi-versi Android : 1.0, 1.1, Cupcakes, Donut, Eclair, Froyo, Gingerbread, Honey Comb, Ice Cream Sandwich, Jelly Bean, Kitkat,Lollipop,Marshmallow,Nougat
